## Creating a blog post
- Post belongs in /_posts
- file name must be: YYYY-MM-DD-Title.md
- Example front meta:
~~~
---
layout: post
title: This is the title
author: Boston Joe Sly
date: 2017-03-12 21:02:00 -0500
published: true
---
~~~
